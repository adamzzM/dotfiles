
My configuration files for the terminal and such 

## Setup 
Install [JetBrains Mono Nerd font](https://www.nerdfonts.com/font-downloads)

Dependingg on weather you're installing this on windows or a unix like system you need to modify the alacritty.toml 

`#import = ["~/.config/alacritty/themes/themes/catppuccin_frappe.toml"] # uncomment if on unix`

`#import = "${ALACRITTY_CONFIG_DIR}/themes/themes/catppuccin_frappe.toml"  # uncomment if on unix `

feel free to explore and change the theme too


general.import = ["C:/Users/adamzzz/AppData/Roaming/alacritty/themes/themes/catppuccin_frappe.toml"]


After you installed alacritty through [the alacritty website](https://alacritty.org/)

copy the alacritty folder to-

    *Windows* -> %APPDATA%\alacritty
    
    *macOS/Linux* -> ~/.config/alacritty
